# Manual de Remediation

**Propósito:** Remediación automática, defensiva y ofensiva controlada.

## Uso
- Se invoca desde el Cognitive Engine o el orquestador.
- Ejecuta scripts de defensa y acciones automáticas.

## Funciones principales
- `Remediate(issue)`: Ejecuta acción defensiva u ofensiva.

## Integración
- Core, cognitive engine, agentes, dashboard.
