# Manual de Agentes Distribuidos

**Propósito:** Ingestión de señales, ejecución de acciones y cobertura de superficie.

## Uso
- Los agentes se despliegan en hosts y dispositivos.
- Recogen señales y ejecutan acciones recomendadas por el Cognitive Engine.

## Comandos principales
- `Agent.CollectSignals()`: Recoge señales.
- `Agent.ExecuteAction(action)`: Ejecuta acción.

## Integración
- Conectados al core y cognitive engine.
