# Manual de IA Táctica y Visual Forensics

**Propósito:** Análisis ofensivo/defensivo con IA local (Ollama + Python) y forense visual.

## Uso
- Ejecutar `ia_tactical.py` para análisis de escaneos y generación de vacunas.
- Ejecutar `visual_forensics.py` para captura y análisis OCR de evidencia visual.
- Ejecutar `visual_forensics_automator.py` para integración y remediación automática.

## Funciones principales
- Predicción de rutas de ataque, generación de reglas de firewall, análisis OCR, remediación inmediata.

## Integración
- Dashboard, remediation, compliance, forensic report.
