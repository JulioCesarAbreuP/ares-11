# Manual de Vacunas y Enforcer

**Propósito:** Scripts de endurecimiento y bloqueo automático.

## Uso
- Scripts generados por IA o analista.
- Ejecutar manualmente o desde módulos de remediación.

## Ejemplo
- `vacuna.ps1`: Bloqueo de vector de ataque.
- `vacuna_isolator.ps1`: Aislamiento de host sospechoso.

## Integración
- Remediation, cognitive engine, forensic analyst.
