# Manual de Sniffers y Forense

**Propósito:** Captura de paquetes, fingerprints y evidencia forense.

## Uso
- Ejecutar `forensic_eye.py` para captura pasiva.
- Ejecutar `forensic_analyst.py` para análisis de persistencia y remediación.
- Ejecutar `generate_forensic_report.py` para reporte forense.

## Funciones principales
- Detección de beaconing, exfiltración, MAC spoofing, TTL mismatch.
- Automatización de vacunas de aislamiento.

## Integración
- Dashboard, compliance, remediation.
