# Vaccines (Scripts de Endurecimiento)

Este directorio almacena los scripts generados por la IA táctica para bloquear vectores de ataque detectados en tiempo real.

- Los scripts se generan automáticamente por `brain/ia_tactical.py`.
- Se ejecutan solo si el riesgo detectado es mayor o igual a 8/10.
- Ejemplo: `vacuna.ps1` (PowerShell)

**Privacidad:** Todo el proceso es local, sin envío de datos a la nube.
