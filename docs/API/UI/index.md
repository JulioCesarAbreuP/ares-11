## Cómo navegar la plataforma

- Utiliza el menú hamburguesa (esquina superior derecha) para acceder rápidamente a cualquier módulo o herramienta.
- Cada interfaz individual describe el propósito, diseño, KPIs y estándares alineados a la doctrina ARES‑11.
- El dashboard táctico centraliza la visualización de riesgos, activos y acciones.
- La documentación doctrinal detalla la arquitectura, controles y flujos de cada componente.

---

## Arquitectura visual

- Estilo minimalista, táctico y empresarial.
- Paneles, tarjetas, badges y gráficos para visualización clara y accionable.
- Integración total con el dashboard táctico y la documentación doctrinal.
