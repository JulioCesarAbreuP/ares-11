# Menú Hamburguesa Empresarial

**Ubicación recomendada:** Esquina superior derecha de la plataforma.

**Comportamiento:**
- Al hacer clic, expande/cuelga un menú lateral o modal.
- Permite colapsar/expandir para navegación rápida.

**Elementos del menú:**
- Inicio
- Arquitectura
- Módulos
- Dashboard táctico
- Documentación doctrinal
- Interfaz (UI)
  - CIS Auditor
  - TAXII Client
  - Risk Engine
  - Remediation Engine
  - Tactical Dashboard
  - Quick Triage
  - Exporter
  - Notifier
  - Live Monitor
  - Remediation Script Generator
  - Whitelist
  - Shodan Check
  - Forensics Packager

**Estilo:**
- Minimalista, táctico y empresarial
- Fondo oscuro, iconos simples, tipografía clara
- Resalta el módulo activo

**Ejemplo conceptual:**


> El menú debe estar disponible en todas las vistas para navegación instantánea.
