# Compliance

**Propósito:** Auditoría, evidencia y reporting alineado a estándares.

**Flujo interno:**
- Recolección de evidencia
- Generación de reportes
- Mapeo a controles

**Integración modular:** Core, agentes, dashboard, remediation.

**Señales y KPIs:** Evidencias, cumplimiento, hallazgos, cobertura.

**Estándares:** CIS 4, NIST AU-6, SABSA Service Layer.

**Resumen ejecutivo:** Compliance automatiza la auditoría y reporting doctrinal de ARES‑11.
