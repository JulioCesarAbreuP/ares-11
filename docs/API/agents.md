# Agentes Distribuidos

**Propósito:** Ingestión de señales, ejecución de acciones y cobertura de superficie.

**Flujo interno:**
- Recolección de señales
- Ejecución de acciones
- Reporte al core

**Integración modular:** Conectados al core y cognitive engine.

**Señales y KPIs:** Señales de host, integridad, cobertura, latencia.

**Estándares:** CIS Control 8, NIST SI-4, MITRE T1021, SABSA Service Layer.

**Resumen ejecutivo:** Los agentes son la extensión táctica y operativa de ARES‑11 en la infraestructura.
