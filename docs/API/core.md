# Core

**Propósito:** Orquestador principal, inicializa y coordina todos los módulos.

**Flujo interno:**
- Inicialización de módulos
- Gestión de ciclo principal
- Recepción y envío de señales

**Integración modular:** Todos los módulos dependen del core para su ciclo de vida.

**Señales y KPIs:** Estado de módulos, latencia, errores críticos.

**Estándares:** CIS Control 6, NIST SI-4, SABSA Service Layer.

**Resumen ejecutivo:** El core es el cerebro táctico y operativo de ARES‑11.
