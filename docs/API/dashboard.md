# Dashboard

**Propósito:** Visualización táctica, KPIs, heatmaps y paneles de control.

**Flujo interno:**
- Recepción de datos
- Renderizado de paneles
- Actualización en tiempo real

**Integración modular:** Core, cognitive engine, agentes, remediation, API.

**Señales y KPIs:** Detecciones, score de riesgo, integridad, acciones automáticas.

**Estándares:** CIS 6, NIST SI-4, SABSA Service Layer.

**Resumen ejecutivo:** El dashboard es la interfaz táctica y ejecutiva de ARES‑11.
