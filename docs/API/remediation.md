# Remediation

**Propósito:** Remediación automática, defensiva y ofensiva controlada.

**Flujo interno:**
- Recepción de alertas
- Ejecución de acciones
- Reporte de estado

**Integración modular:** Core, cognitive engine, agentes, dashboard.

**Señales y KPIs:** Acciones ejecutadas, tiempo de respuesta, éxito/fracaso.

**Estándares:** CIS 10, NIST IR-4, SABSA Service Layer.

**Resumen ejecutivo:** Remediation automatiza la respuesta táctica ante incidentes y hallazgos.
