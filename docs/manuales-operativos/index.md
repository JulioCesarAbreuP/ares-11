# Índice Maestro de Manuales Operativos ARES‑11

Bienvenido a la colección de manuales operativos de ARES‑11. Aquí encontrarás la guía táctica, doctrinal y empresarial para operar, auditar y mantener cada módulo y herramienta del sistema.

## Propósito de los Manuales
- Estandarizar procedimientos y flujos operativos
- Facilitar la integración y el despliegue
- Mejorar la auditoría, la respuesta y la resiliencia

## Navegación y Estructura
- Cada manual aborda: propósito, SOP, flujos, KPIs, integración, escenarios, estándares y resumen ejecutivo
- Usa el menú hamburguesa para navegar por módulos
